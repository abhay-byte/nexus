## Changelog

### Major Features

- **feat: Headless server mode (`nexus-headless`)**
  - Run Nexus without a desktop window — accessible from any browser on your local network
  - HTTP API on port `7878` for REST endpoints (projects, kanban, git, agents, filesystem)
  - WebSocket on port `7879` for real-time terminal streaming
  - IP-based access control — only localhost + local network IPs allowed
  - Extra IPs configurable via `NEXUS_ALLOW_IPS` env var
  - All data persisted to `nexus_web_state.json`
  - Nice startup banner showing all accessible URLs

- **feat: Unified browser/desktop API layer**
  - Frontend automatically detects Tauri vs browser mode
  - All Tauri commands have HTTP proxy endpoints
  - WebSocket bridges Tauri event listeners in browser mode
  - Terminal spawns via WebSocket with full PTY support

- **feat: Browser-mode file tree**
  - Project directory panel works in browser via HTTP `/api/fs/read-dir`
  - File preview works via HTTP `/api/fs/read-text-file`
  - No Tauri plugin-fs required in browser

- **feat: Mobile portrait mode support**
  - Sidebar becomes slide-out overlay with hamburger menu on phones
  - Project directory panel becomes full-width overlay on mobile
  - Kanban board columns horizontally scrollable with scroll-snap
  - Terminal pane grid stacks vertically on narrow screens
  - Settings panel stacks vertically on mobile
  - Agent selector uses 2-column grid on mobile
  - Status bar and toolbars wrap gracefully on small screens
  - Desktop landscape mode completely unchanged

### Fixes

- **fix: Directory picker showing "Select File" on Linux**
  - `isTauri()` now uses the official `window.isTauri` check from `@tauri-apps/api/core`
  - Previously checked removed `window.__TAURI__` global, causing fallback to browser file input
  - Directory picker now correctly uses native Tauri dialog on desktop

- **fix: Terminal broken with "state not managed" error**
  - Fixed `State<'_, AppState>` mismatch — commands now expect `State<'_, Arc<AppState>>`
  - Matches `main.rs` which wraps state in `Arc` before `.manage()`

- **fix: Headless server returns `{"error": "Not Found"}`**
  - `find_dist_dir()` now searches additional locations: compile-time project dir, `~/.local/share/nexus/dist`
  - Ensures frontend assets are found regardless of where binary is installed

- **fix: Headless server "Address already in use"**
  - Added `kill_port_occupiers()` that kills existing processes on the target port before binding
  - Works on Linux (fuser/lsof/proc fallback) and Windows (netstat + taskkill)

- **fix: Terminal spawn timeout in browser mode**
  - WebSocket spawn no longer waits for confirmation event that never fires
  - Resolves "WebSocket spawn timeout" error after terminal opens

- **fix: Tauri plugin crashes in browser**
  - All Tauri plugin imports are now lazy-loaded
  - Browser mode no longer crashes on module load

- **fix: Project/kanban persistence in headless mode**
  - Server saves state to disk on every mutation
  - State automatically loaded on server restart
