Multi-project AI agent workspace with integrated terminal.

## Downloads

### Linux
| Format | File | Size |
|---|---|---|
| Debian/Ubuntu (x86_64) | `Nexus_0.1.4_amd64.deb` | 4.6 MB |
| RHEL/Fedora (x86_64) | `Nexus-0.1.4-1.x86_64.rpm` | 4.6 MB |
| Tarball (x86_64) | `Nexus_linux_x64.tar.gz` | 4.4 MB |
| Raw binary (x86_64) | `nexus-linux-x86_64` | 15 MB |

**Installation (Debian/Ubuntu):**
```bash
sudo apt install ./Nexus_0.1.4_amd64.deb
```

**Installation (RHEL/Fedora):**
```bash
sudo dnf install ./Nexus-0.1.4-1.x86_64.rpm
```

**Installation (Tarball):**
```bash
tar -xzf Nexus_linux_x64.tar.gz
cd nexus-linux-x64
./nexus
```
